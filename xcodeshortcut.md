
Xcode Shortcut 
================


Control pages
---------

Move forward and backward
** cmd  +  ctrl  +  <-or->**



Text edit
----------

Delete a line behind cursor
** ctrl  +  K **

One word
** alt  +  shift  +  <-or->  **

Big one word
** ctrl  +  shift  +  <-or->  **

Function  remark
**  Cmd + Option + /    **
